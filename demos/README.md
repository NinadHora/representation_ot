# Demos (planned)

Synthetic examples reproducible without access to any external dataset.

Planned first demo, `synthetic_coherence.py`: a Gaussian mixture in which one
category is constructed to be geometrically incoherent (its members drawn from
distinct modes shared with other categories). The coherence pipeline of
`case_studies/facial_embeddings/phase2_coherence.py` must detect the
incoherent category. This provides an executable specification of the method
for readers without CCv2 access.
